# ECG-FM labeler config (MIMIC-IV-ECG)

Pattern labeler configuration used by **labels/scripts/create_labels_ecgfm.py**, from the [ECG-FM](https://github.com/bowang-lab/ECG-FM) official repository. Contains **\*.json** (entity/descriptor templates, patterns, connectives) and **label_def.csv** (17 diagnostic labels for fine-tuning).
